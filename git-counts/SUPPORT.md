# Git Counts Support

## Supported Environment

Git Counts supports VS Code 1.125.0 or later with the built-in Git extension enabled and a local Git repository open in the workspace.

## Before Requesting Support

1. Confirm that VS Code's built-in Git extension is enabled.
2. Confirm that the repository appears in the Source Control view.
3. Run **Developer: Reload Window** and try the action again.
4. Check whether Git Counts displayed a compatibility message naming a missing VS Code command.

## Requesting Support

Use the **Q & A** section on the Git Counts Marketplace page. Include:

- Git Counts version.
- VS Code version from **Help: About**.
- Operating system.
- Whether the workspace contains one repository or multiple repositories.
- The exact error message and the steps that produced it.

Do not include private repository names, paths, source code, credentials, or other sensitive information.

## Privacy

Git Counts has no telemetry and makes no network requests. Support information is only shared when you choose to post it to Marketplace Q & A.
