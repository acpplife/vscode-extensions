# Changelog

All notable changes to Git Counts are documented in this file.

## 0.1.4

- Added a Marketplace icon, search keywords, banner styling, support information, and complete English metadata.
- Documented features, usage, repository selection, compatibility, privacy, and licensing.
- Updated all user-facing labels and messages to English for the public release.

## 0.1.3

- Reduced selection flicker while switching SCM groups.
- Collapsed other groups and expanded only the selected group after a status bar click.

## 0.1.2

- Fixed repository lookup failures caused by different Git API wrapper instances in multi-root workspaces.

## 0.1.1

- Improved compatibility checks when internal list navigation commands are unavailable.

## 0.1.0

- Added separate status bar entries for staged and unstaged file counts.
